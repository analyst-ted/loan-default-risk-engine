# loan-default-risk-engine
Deep Learning pipeline to predict loan default risk and expected loss on LendingClub data

## 📂 Dataset Setup

The dataset is not included in this repository due to size.

1. Download from [Kaggle — LendingClub Loan Data](https://www.kaggle.com/datasets/wordsforthewise/lending-club)
2. Place the file at: `data/raw/lending_club_loan_two.csv`
3. Run notebooks in order starting from `01_data_cleaning.ipynb`
